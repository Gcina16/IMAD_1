# Social Spark – IMAD5112 Assignment 1

## Description
Social Spark is a native Android application developed using Kotlin in Android Studio.
The app suggests small social interactions ("sparks") based on the time of day entered by the user.

The purpose of this app is to help users stay socially connected during busy days.

---

## Features
- Time-of-day based social suggestions
- Input validation and friendly error messages
- Reset functionality
- Logging using Logcat
- GitHub version control
- GitHub Actions for automated building

---

## How It Works
The user enters a time of day (e.g. Morning, Afternoon, Dinner).
The app uses Kotlin IF statements to determine which social spark to display.

---

## Example Inputs
- Morning → Send a "Good morning" message
- Afternoon → Share a meme or link
- Dinner → Call a friend or relative

---

## Error Handling
- Empty input prompts user feedback
- Invalid inputs display corrective messages
- Logcat logs used for debugging purposes

---

## GitHub Actions
This project uses GitHub Actions to automatically build the app on every push.
